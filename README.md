# Atlassian 프로젝트

이 프로젝트는 React 애플리케이션에서 Atlassian Design System 컴포넌트를 사용하여 구현한 산출물입니다.

## 시작하기

### 필수 조건

- Node.js (버전 20.13.1)
- Yarn (버전 1.22.19)

### 설치

```text
# 1. 레포지토리 클론하기:

git clone https://github.com/siwon-dev-git/project-atlassian-osc.git

# 2. 프로젝트 디렉토리로 이동하기:

cd atlassian-project

# 3. 의존성 설치하기:

yarn install

# 4. 개발 서버를 시작하기:

yarn dev

```

## 사용된 기술

- React 16.14
- Atlassian Design System 컴포넌트
- Webpack 5
- TypeScript
- i18next (국제화)
